What I learned
Although this was categorized as a ‘Junior’ challenge, I took this challenge because I wanted more practice on how to effectively utilize the Bootstrap grid system. Realizing the long term benefits mastering this topic would provide, I believe this project would be a great opportunity to not only reinforce my understanding of Bootstrap but more how to work with grid containers. I’ve definitely become more confident in my ability to recognize grid layouts and how to translate that to code. I also gain a better appreciation for Bootstrap as it relates to saving time, its limitations and syntax. 

Continued development
While working on this project, I still encountered challenges when it came to customizing the layout of some elements on the grid using Bootstrap. This is a key area that I’ve identified I still need to work on.
